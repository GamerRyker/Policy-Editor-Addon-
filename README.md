# Policy-Editor-Addon-
For the people who ended up using the Policy Editor Script, but some / all of the required extensions were not installed.
STEPS:
1. Open the VT2 Developer Shell (Control + ALT + F2)
2. Run the following command then press enter: "cd"
3.  Run the following command then press enter: "cd /etc/opt/chrome/policies/"
4.  Run the following command then press enter: "cd managed" (or add "managed" to the command above this one,) ( cd /etc/opt/chrome/policies/managed )
5.  Run the following command then press enter: "curl -O https://raw.githubusercontent.com/GamerRyker/Policy-Editor-Addon-/refs/heads/main/PolicyEditorV2.sh"
6.  Run this command ONLY if you have ran the other Policy Editor Script: cat newpolicy.json > policy.json

We are not responsible for any damages or issues this may cause to your Chromebook! Use at your own RISK.

For those of you who have Read&Write, you will have to edit the script to include Read&Write as a force installed extension.
